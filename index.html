First time?

Installing an APK

Android blocks installs from outside the Play Store by default. Three steps and you're set.

01

Allow this source

Open Settings → Apps → Special access → Install unknown apps, choose your browser or Files app, and turn it on.

02

Download the APK

Tap Download on the app you want. Your browser will save it, usually to the Downloads folder.

03

Open and install

Open the downloaded file from your notifications or Files app, then confirm the install prompt.

Only install APKs you got from a source you trust — since these come straight from my own server rather than an app store.
